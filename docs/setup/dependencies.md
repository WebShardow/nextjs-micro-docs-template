# รายการ Dependencies และ Scripts

## การติดตั้ง Dependencies

เราต้องติดตั้งแพ็กเกจที่จำเป็นสำหรับ Next.js เพื่อให้สามารถประมวลผล MDX ได้

```Bash
# ติดตั้ง MDX Loader, Remark และ Rehype (สำหรับ Markdown Parsing และ Plugin)
npm install @next/mdx @mdx-js/loader @mdx-js/react
npm install --save-dev @types/mdx
```
